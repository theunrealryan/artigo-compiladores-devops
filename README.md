# Análise de Paradigmas de Programação e Gerenciamento de Memória

Este repositório contém o estudo de caso prático e os códigos-fonte utilizados no artigo **"Análise de Paradigmas de Programação e Gerenciamento de Memória no Ecossistema DevOps e Cloud Native"**, desenvolvido para a disciplina de Compiladores e Interpretadores.

## 🎯 Objetivo
Analisar o impacto da escolha entre linguagens interpretadas (ex: Python) e compiladas (ex: C) em relação ao tempo de execução e consumo de memória (Heap), simulando cenários críticos de arquiteturas *Cloud Native* e contêineres.

## 🛠️ Tecnologias Utilizadas
* **Jupyter Notebook:** Orquestração do experimento e geração de gráficos.
* **Python 3:** Representando as linguagens interpretadas (usando `memory_profiler`).
* **C (GCC):** Representando as linguagens compiladas nativamente (Ahead-of-Time).
* **Matplotlib / Seaborn:** Para a visualização dos dados.
